# example-skill

Template skill demonstrating the agentskills.io layout used in this repository.

This README is the human-facing entry point for the skill. The agent-facing
contract lives in [`SKILL.md`](SKILL.md); Codex/OpenAI exposure is configured in
[`agents/openai.yaml`](agents/openai.yaml).

## Install

Copy this directory into your agent's skills location, e.g. for Claude Code:

```sh
cp -r example/example-skill ~/.claude/skills/example-skill
```
