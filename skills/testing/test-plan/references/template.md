| # | リスク項目 | 発生可能性 | 影響度 | リスクレベル | 根拠 |

| R1 | <期待どおり動かないと何が起きるか> | 高/中/低 | 高/中/低 | 高/中/低 | <なぜその可能性・影響度か（調査事実）> |
| R2 | | | | | |

- 評価基準: リスクレベル = 発生可能性 × 影響度（高い順に優先）。
- 計上前に該当経路のコードで発生し得ることを確認する（到達しない防御的分岐を実リスクにしない）。
